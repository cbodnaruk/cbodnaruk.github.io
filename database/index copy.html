TH Epistemic Database v1

This is a database of summaries of epistemic marking systems in Trans-Himalayan languages. This is a living project that I will continue to update and add to as time goes on. For any errors or corrections, please email me.

Clicking on a column will expand it to show all its contents, clicking again will return it to its default size. Double clicking on a row below will bring up a table with only the data from that language for ease of reading. Please do not cite this database - use the references provided to find the original data and analysis.
